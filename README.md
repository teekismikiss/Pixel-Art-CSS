
**08 Ene 2026** # Pixel Art CSS - Espada Box-Shadow

Arte pixelado de espada creado **exclusivamente con CSS puro** usando `box-shadow` para simular píxeles. Proyecto del curso **Desarrollo de Aplicaciones con Tecnología Web**.

## Características Técnicas
- **CSS Variables** (`:root`): 10 colores con shading metálico preciso (acero cian/azul, oro degradado, empuñadura marrón).
- **Box-shadow avanzado**: 70+ sombras organizadas por 21 filas para diseño pixel-perfect.
